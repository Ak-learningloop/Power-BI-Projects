HR Analytics Dashboard 📊

📌 Project Overview
This project presents an HR Analytics Dashboard designed to help organizations monitor workforce trends, identify attrition patterns, and support data-driven HR decisions. The dashboard provides insights into employee distribution, attrition rates, job satisfaction, and demographic segmentation.

🎯 Business Requirements
The dashboard was built to address the following key business questions:

Workforce Overview
Total number of employees
Active employees
Average employee age
Attrition Analysis
Total attrition count
Attrition rate (%)
Attrition distribution across departments
Demographic Insights
Employee distribution by age groups
Gender-based workforce segmentation
Attrition trends across age groups and gender
Education & Role Analysis
Attrition in the education field
Job satisfaction levels across different roles
Decision Support
Identify high-risk attrition segments
Understand workforce composition
Improve retention strategies

🛠️ Dashboard Components & Visualizations
1. KPI Cards

Used to display high-level metrics:
👉 These provide a quick snapshot for executives.

2. Department-wise Attrition (Donut Chart)
Visualizes attrition distribution across:
R&D
Sales
HR
👉 Helps identify departments with the highest employee turnover.

4. Age Group Analysis (Stacked Bar Chart)
Shows the number of employees by:
Age bands (Under 25, 25–34, etc.)
Gender (Male vs Female)
👉 Useful for workforce planning and demographic analysis.

4. Job Satisfaction Matrix (Matrix/Table)
Displays satisfaction ratings (1–4) across job roles
Includes totals for each role
👉 Helps HR understand employee engagement at the role level.

5. Education Field Attrition (Bar Chart)
Shows attrition counts by educational background:
Life Sciences
Medical
Marketing
Technical Degree, etc.
👉 Identifies which educational backgrounds have higher attrition.

6. Attrition by Age & Gender (Donut Charts)
Displays attrition split for each age group
Highlights gender-based attrition percentages
👉 Helps detect vulnerable workforce segments.
